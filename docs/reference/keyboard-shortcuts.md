# Keyboard Shortcuts

All keyboard shortcuts for the TUI components.

## Dashboard (`ws dashboard`)

### Navigation

| Key | Action |
|---|---|
| `j` / `â†“` | Select next workstream |
| `k` / `â†‘` | Select previous workstream |

| Key | Action |

## Diff Viewer (`ws diff <name>`)

### Panel Navigation

| Key | Action |
|---|---|
| `Tab` / `h` / `l` | Switch between file list and diff panels |

|---|---|

| Key | Action |
|---|---|

| Key | Action |
|---|---|

| `q` | Quit viewer |
